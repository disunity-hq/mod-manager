# mod-manager
Disunity Mod Manager for Unity games

# How to install

  - ```git clone https://github.com/disunity-hq/mod-manager.git```
  - ```npm install```

# How to run
### Development version
- ```npm run dev```
Initiates a local webserver and fires up electron.
### Packaged version
- ```npm run build```
Outputs a build to the projectdir/build/ folder
